Privacy Policy

Last updated: January 24, 2026

TL;DR: Chronos does NOT collect any personal data. The app runs entirely offline on your device. No files are uploaded, transmitted, or stored remotely.

1. Data Collection

Chronos does NOT collect any personal information.

The application is designed to operate completely offline and locally on your device. We do not:

  • Collect user names, email addresses, or contact information
  • Track your usage behavior or analytics
  • Access your location or device identifiers (except for trial management, see below)
  • Upload, transmit, or store any of your files remotely

2. File Access and Processing

Chronos requires access to your files and folders to perform its core functionality: merging text files into a single output.

File access is limited to:

  • Files and folders you explicitly select via:
    • Windows Explorer context menu (right-click)
    • Drag-and-drop into the application
    • File picker dialog within the app
  • Reading file contents for extraction and merging
  • Creating a single output text file with merged content

What we DO NOT do with your files:

  • Upload files to any server or cloud service
  • Share files with third parties
  • Scan or access files you haven't explicitly selected
  • Store copies of your files beyond the output file you create

3. Trial System and Local Data Storage

Chronos includes a free trial system that allows 3 free extractions before requiring a license purchase.

Trial data stored locally:

  • Location: %LOCALAPPDATA%\Chronos\trial.db (SQLite database)
  • Data stored:
    • Number of extractions performed (integer counter)
    • Trial expiration timestamp (date/time)
    • Device identifier (anonymous hash, used only to prevent trial reset)
  • Purpose: Tracking trial usage to enforce the 3-extraction limit
  • Transmission: This data is NEVER transmitted off your device
  • Deletion: You can delete this file manually, or it will be removed when you uninstall the app

4. License Management

Depending on your platform and purchase method, Chronos uses different licensing systems:

4.1 Microsoft Store (Windows 11)

If you purchased Chronos via the Microsoft Store, all payment processing and license management is handled by Microsoft.

  • What Microsoft collects: Refer to Microsoft's Privacy Statement
  • License verification: The app queries the Microsoft Store API locally to verify your purchase
  • No personal data: Chronos does not receive or store your Microsoft account information

4.2 License Key System (Future Implementation)

Note: We are planning to introduce an alternative licensing system that allows purchasing license keys directly from our website or third-party platforms (e.g., Gumroad, Paddle).

When implemented, this system will:

  • Allow you to enter a license key purchased elsewhere
  • Validate the key via a secure HTTPS API call (one-time only)
  • Store the validated license key locally on your device (encrypted)
  • Operate offline after initial validation

Data transmitted during license key validation:

  • License key (alphanumeric string)
  • Device identifier (anonymous hash, to prevent key sharing abuse)
  • App version number

No personal information (name, email, payment details) is transmitted by the Chronos app. If you purchase a license key from our website or a third-party platform, their privacy policies apply to that transaction.

5. Network Access and Connectivity

Current version: Chronos operates entirely offline and does NOT connect to the internet.

Future updates may include optional network features:

  • Update checks: The app may check for new versions via Microsoft Store API (Windows 11) or our update server
  • License key validation: One-time HTTPS call to validate a license key (if using external licensing)
  • Crash reporting (opt-in only): Anonymous crash logs to help us improve stability (feature not yet implemented)

All network features will be:

  • Clearly communicated to users
  • Optional (where applicable)
  • Secure (HTTPS only)
  • Anonymous (no personal data transmitted)

6. Third-Party Services

Current version: Chronos does NOT integrate with any third-party services, SDKs, or analytics platforms.

Potential future integrations (not yet implemented):

  • Payment processors: If you purchase a license key via our website, we may use payment processors like Stripe, PayPal, or Gumroad. Their privacy policies apply to those transactions.
  • Cloud storage (optional): Future versions may allow optional integration with cloud services (OneDrive, Google Drive) to sync settings or presets. This would be opt-in only.

We will update this privacy policy if any third-party services are integrated.

7. Windows 10 vs Windows 11 Differences

7.1 Windows 11 (Microsoft Store MSIX)

  • Context menu integration is sandboxed within the MSIX package
  • File access is governed by Windows 10/11 privacy permissions (broadFileSystemAccess capability)
  • License managed by Microsoft Store
  • Updates delivered automatically via Microsoft Store

7.2 Windows 10 (Standalone Edition - Not Yet Available)

Note: A Windows 10 standalone version is planned for future release. When available:

  • Context menu integration will be installed via registry (requires Administrator privileges once)
  • File access is unrestricted (standard desktop application)
  • License will use external license key system (see section 4.2)
  • Updates will be manual download or auto-update check (to be determined)

8. Children's Privacy

Chronos is a productivity tool designed for general audiences aged 3+ (as rated in the Microsoft Store).

  • The app does not knowingly collect personal information from children under 13
  • No user accounts or profiles are created
  • No social features or user-generated content sharing

9. Data Retention and Deletion

Local data stored by Chronos:

  • Trial database: %LOCALAPPDATA%\Chronos\trial.db – retained until app uninstall or manual deletion
  • License key: Encrypted and stored locally – retained until app uninstall
  • App settings: Preferences and configuration – retained until app uninstall

To delete all Chronos data:

  1. Uninstall the app via Windows Settings → Apps → Chronos → Uninstall
  2. (Optional) Manually delete the data folder: %LOCALAPPDATA%\Chronos\

Since Chronos does not transmit data to any servers, there is no remote data to delete.

10. Security

We take reasonable measures to protect data stored locally by Chronos:

  • License keys: Encrypted using industry-standard encryption (AES-256)
  • Network communication: All future network features will use HTTPS only
  • Code signing: The application executable is signed to prevent tampering (Microsoft Store version)

However, as Chronos operates entirely on your local device, the security of your data depends primarily on your device's security settings and your Windows user account protections.

11. International Users and Data Transfers

Since Chronos processes all data locally on your device and does not transmit data to external servers:

  • No international data transfers occur
  • GDPR, CCPA, and other privacy regulations are satisfied by design (no data collection)
  • The app can be used globally without privacy concerns

Future licensing system: If license key validation is implemented, the validation server may be hosted in the EU or USA. Only anonymous data (license key + device hash) will be transmitted.

12. Your Rights

Under various privacy laws (GDPR, CCPA, etc.), you have rights regarding your personal data. However, since Chronos does not collect or store personal data remotely:

  • Right to access: All data is stored locally on your device and accessible to you
  • Right to deletion: You can delete all local data by uninstalling the app
  • Right to data portability: Not applicable (no remote data)
  • Right to object: You control all file access via explicit user actions

If you have questions about your data, please contact us (see section 14).

  • You can always find the latest version at: https://chronos.dev/privacy

Continued use of Chronos after changes indicates your acceptance of the updated policy.
